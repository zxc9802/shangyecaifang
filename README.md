# Elite Business Interviewer Skill

## 📖 简介

这是一个**顶级商业访谈记者**技能，能够通过动态多轮深度访谈挖掘用户的商业故事，并自动生成带配图的吸睛公众号文章。

## ✨ 核心特性

- 🎙️ **动态访谈**：自由发挥追问，自行判断素材饱满度
- 📝 **自动写作**：一次性输出完整文章（3个备选标题 + ≤1000字正文）
- 🎨 **自动配图**：在文章合适位置生成精美配图
- 🔄 **风格进化**：每次生成的文章成为下次的风格参考
- 🤫 **零中断**：全程自动化，不询问确认

## 📂 文件结构

```
├── skills/
│   └── elite-business-interviewer/
│       ├── SKILL.md              # 技能定义文件
│       ├── VALIDATION_MATRIX.md  # 测试验证矩阵
│       └── README.md             # 本文件
├── articles/                     # 生成的文章和配图存放处
│   ├── YYYY-MM-DD_标题.md
│   └── YYYY-MM-DD_标题_01.png
└── ...
```

## 🚀 使用方式

只需对 Claude 说以下任意一句：

- "采访我"
- "挖掘我的故事"
- "商业访谈"
- "写一篇关于我创业的公众号文章"
- "帮我讲好我的创业故事"

Claude 会自动进入访谈模式，完成后输出完整文章。

## 📋 输出示例

```markdown
## 📌 备选标题

1. 从濒临倒闭到年入千万：一个AI教育创业者的生死180天
2. "那天银行账户只剩47块，我把团队叫到了天台"
3. 没人看好的赛道，他却押上了全部身家

---

## 📝 正文

账户余额：47元。
...
[配图]
...
```

## ⚙️ 配置项

| 配置 | 默认值 | 说明 |
|------|--------|------|
| 文章长度 | ≤1000字 | 超过自动精简 |
| 备选标题 | 3个 | 悬念型/情感型/反差型 |
| 配图数量 | 1-2张 | 根据故事情绪自动生成 |
| 开场风格 | 直接专业型 | 以核心业务和挑战开场 |

## 📌 注意事项

1. 首次运行时 `articles/` 文件夹为空，会使用内置默认风格
2. 后续运行会自动参考最新一篇文章的风格
3. 配图依赖 Claude 的 generate_image 能力
